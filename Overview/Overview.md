# 📌 Project Overview

This project focuses on **Networking and Security Operations using SIEM, Digital Forensics, and Traffic Analysis**, simulating real-world SOC (Security Operations Center) activities. The aim is to detect, analyze, and investigate security incidents using multiple data sources.

The project involves collecting and analyzing logs using SIEM concepts, capturing and inspecting network traffic using Wireshark, and performing basic forensic analysis to identify potential compromises. By correlating logs, traffic data, and forensic evidence, the project demonstrates how security analysts detect and respond to threats.

A structured methodology was followed, including environment setup, log monitoring, traffic analysis, incident detection, and forensic investigation. Special attention was given to identifying indicators of compromise and understanding attacker behavior.

This project highlights the importance of centralized monitoring, real-time detection, and post-incident investigation in cybersecurity. It also provides insights into how SOC teams operate to protect organizational infrastructure.

Overall, this work builds strong foundational skills required for roles such as SOC Analyst, Threat Hunter, and Incident Responder.
